# Vivek Lanke Data Analyst Portfolio

Modern responsive portfolio for Vivek Lanke, a fresher Data Analyst and AI/ML student. The site highlights Python, SQL, Excel, Power BI, analytics projects, certifications, education, achievements, resume download, and contact details.

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Motion
- Lucide React

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
```

The project is configured for static export with `output: "export"`, which works well for Vercel and GitHub Pages.
